**Corner Pin Effect Filter**

A Lua script for OBS that adds a 4-point perspective warp (Corner Pin) filter. Transform any source by independently dragging its corners to create 3D perspective effects.

### **Installation**
1. Copy `filter-cornerpin.lua` to your OBS scripts folder.
2. In OBS, go to **Tools** > **Scripts**.
3. Click **+** and select the file.
4. Add the "Corner Pin" filter to any Source or Scene.

### **Features**
*   Full Perspective Warp transformation.
*   Precise slider controls for all four corners.
*   Optimized shader-based rendering.
